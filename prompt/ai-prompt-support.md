Lần đầu với 1 AI / 1 workspace:

1. Dán Prompt chuẩn (luật cho AI)
2. Chọn 1 trong số cách dưới tùy AI
   2.1. Import Spec / Contract / base.css + chat "Apply..."
   2.2. Link GitHub
   2.3. Chat:
   Use my personal Design System from this repository (or github link).
   Refer to CURRENT.md for the active configuration.

3. Dùng Daily Prompt + task
   Ví dụ: [Daily Prompt] + Create an e-commerce product listing page.

Khi bạn update Design System trên GitHub:

1. Sau khi commit:
   Spec / Contract / base.css
   Update CURRENT.md
2. Chỉ cần nói với AI:
   Design System updated. Re-read CURRENT.md.
